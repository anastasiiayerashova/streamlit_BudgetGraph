# 💸 AI-Асистент Особистих Фінансів (Smart Finance Agent)

Веб-застосунок на базі штучного інтелекту для зручного відстеження та аналізу особистих фінансів за допомогою природної мови. Користувач може спілкуватися з розумним агентом у форматі чату, додавати витрати на льоту, встановлювати ліміти та миттєво бачити аналітику на інтерактивній панелі керування (Dashboard).

Проєкт реалізовано за допомогою сучасного підходу **Agentic AI** на базі фреймворку **LangGraph** та LLM **Gemini 2.5 Flash**, з візуалізацією інтерфейсу через **Streamlit**.

---

## ✨ Основні можливості та функції

### 💬 Розумний чат-інтерфейс (Головна область)

- **Природна мова:** Введення витрат у довільній формі (наприклад, _"Запиши 250 грн на таксі до офісу"_ або _"Встанови ліміт на цей місяць 12000 гривень"_).
- **Гнучкість режимів:**
  - `🛠️ Агент з інструментами` — повноцінний ШІ-помічник із доступом до бази даних, кастомною логікою збереження стану та викликом інструментів (Function Calling).
  - `💬 Звичайний чат` — режим вільного спілкування (без взаємодії з БД) з можливістю налаштування температури генерації та редагуванням системного промпту в реальному часі.
- **Прозорий дебаг:** Можливість увімкнути відображення структури виклику інструментів ШІ (Tool Calls) для розробника або просунутого користувача.

### 📊 Інтерактивний Dashboard (Бічна панель / Sidebar)

- **Метрики фінансового стану (`st.metric`):**
  - **Поточний ліміт:** Відображення встановленого бюджету.
  - **Усього витрачено:** Автоматичний підрахунок суми всіх транзакцій у системі.
  - **Залишок:** Динамічний розрахунок доступних коштів з кольоровою індикацією (`delta` змінює колір на червоний у разі перевищення).
  - **Найбільша категорія:** Автоматичне визначення категорії, на яку витрачено найбільше коштів.
- **Прогнозування витрат на кінець місяця:** Розумний алгоритм розраховує середньодобові витрати на поточний момент та прогнозує фінальну суму на кінець місяця, попереджаючи про ризики виходу за ліміт.
- **Візуалізація ліміту (`st.progress`):** Прогрес-бар використання бюджету з текстовими попередженнями при досягненні 80% та 100% ліміту.
- **Таблиця транзакцій з фільтрацією (`st.selectbox` + `st.dataframe`):** Відображення останніх 5 транзакцій з можливістю інтерактивного фільтру за категоріями.
- **Кругова діаграма розподілу (`Altair / st.altair_chart`):** Інтерактивний графік, який динамічно реагує на вибрані фільтри та показує частку кожної категорії у виділеному бюджеті.

### 💾 Експорт та керування даними

- **Експорт у JSON:** Збереження повної історії чату, транзакцій та налаштувань ліміту в один клік.
- **Експорт в Excel (`openpyxl`):** Генерація структурованого та чистого фінансового звіту у форматі `.xlsx` для подальшого аналізу.
- **Очищення сесії:** Можливість миттєво скинути стан графа, очистити пам'ять (`thread_id`) та почати облік заново.

## 🏗️ Архітектура Агента (LangGraph & Tools)

Застосунок використовує граф станів (`StateGraph`), що дозволяє агенту зберігати контекст розмови та стан гаманця користувача (`expenses` та `monthly_limit`) між повідомленнями завдяки `InMemorySaver`.

### Схема графа:

1. `START` ➡️ **`agent` (Вузол LLM)**: Аналізує запит користувача, враховуючи системні інструкції та поточний стан.
2. **`router` (Умовний перехід)**:
   - Якщо модель вирішила викликати інструмент ➡️ перехід на вузол **`tools`**.
   - Якщо користувач попросив текстовий звіт/аналітику ➡️ перехід на вузол **`reporter`**.
   - Якщо відповідь кінцева ➡️ `END`.
3. **`tools` (Кастомний вузол виконання)**: Перехоплює аргументи ШІ, оновлює стан графа (`AgentState`), виконує бізнес-логіку та повертає результат моделі, зациклюючи хід назад на `agent`.

### Реалізовані інструменти (Tools):

- `add_expense(amount, category, description)` — валідує та додає нову транзакцію до пам'яті.
- `total_spent()` — повертає агреговану суму витрат користувача.
- `summary_by_category()` — повертає актуальний зріз стану (ліміт та масив витрат). Викликається ШІ перед проведенням будь-яких розрахунків чи аналітики.
- `set_limit_tool(new_limit)` — оновлює глобальний місячний ліміт у стані системи.

## 🚀 Швидкий запуск

### 1. Клонування репозиторію

````bash
git clone [https://github.com/anastasiiayerashova/streamlit_BudgetGraph](https://github.com/anastasiiayerashova/streamlit_BudgetGraph)
```

### 2. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 3. Налаштування API Ключа

Створіть директорію .streamlit та файл secrets.toml у корені проєкту:

```bash
mkdir .streamlit
touch .streamlit/secrets.toml
```
Додайте туди свій ключ від Google AI Studio:

```bash
GOOGLE_API_KEY = "ваш_api_ключ_gemini"
```
### 4. Запуск застосунку
```bash
streamlit run app.py
```
Веб-інтерфейс автоматично відкриється у вашому браузері за адресою http://localhost:8501
